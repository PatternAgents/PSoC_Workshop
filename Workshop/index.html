Introduction to Programmable Hardware

An introduction and overview of creating simple programmable hardware projects using the Cypress Programmable System on Chip (PSoC4). Training and working project examples using the PSoC Creator IDE.

Workshop Objectives :

  • Overview of Programmable System on Chip (Cypress PSoC 3/4/5)
  • Learn to use the PSoC Creator IDE for hardware configuration, software generation, program development, and debugging
  • Be able to build and boot load a project on PSoC4

Workshop Length :

  • One Hour for Introduction/Overview
  • One Hour to Three Hours for Hand's On Labs
  • Total : 2 to 4 hours

Required Hardware :

  1. Windows Compatible Computer with USB port
  2. Cypress CY8CKIT-049 (PSoC4) Development board
